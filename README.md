# EnSF-LR

Code for the paper:

**A Two-Step Ensemble Score Filter for Data Assimilation in Partially Observed Nonlinear Systems**

This repository contains experiments for the Ensemble Score Filter with Linear Regression (EnSF-LR), evaluated on Lorenz-63 and Lorenz-96 systems with sparse linear and nonlinear observation operators.

## Experiment Folders

- `Linear_lorenz63/`: Lorenz-63 experiments with sparse linear observations.
- `Linear_lorenz96/`: Lorenz-96 experiments with sparse linear observations.
- `Nonlinear_lorenz63/`: Lorenz-63 experiments with sparse nonlinear observations.
- `Nonlinear_lorenz96/`: Lorenz-96 experiments with sparse nonlinear observations.
- `figures/`: Output plots and diagnostics for each experiment group.
- `Rev_SDE_vanilla.py`: Shared reverse SDE implementation.

Data files are generated by the experiment scripts and are not stored in the repository by default.

## Installation

Create and activate a Python environment, then install the dependencies:

```bash
pip install -r requirements.txt
```

For Intel macOS environments using `torch==2.2.2`, keep NumPy below version 2:

```bash
pip install numpy==1.26.4 torch==2.2.2
```

## Reproducing Experiments

Run the Python scripts inside each experiment folder. For example:

```bash
cd Nonlinear_lorenz63
python generate_data_nonlinear_lorenz63.py 50
python EnSF_nonlinear_lorenz63_0.py 50
```

Use the corresponding Lorenz-63 or Lorenz-96 folder depending on the observation setting you want to reproduce.
